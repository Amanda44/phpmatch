# Installation de PHPUnit
## Installer PHP7
### Pour Windows :
http://windows.php.net/download/

### Pour debian :
sudo apt-get remove php5
https://angristan.fr/installer-php-7-debian-8-jessie-depot-dotdeb/

### Pour ubuntu :
sudo apt-get remove php5
sudo apt-get install -y php7.0


## Installer phpunit
### Linux
sudo apt-get install phpunit
